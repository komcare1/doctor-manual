DOCTOR APP MANUAL

INTRODUCTION

This manual is an official document of KompleteCare™, a digital product of Sevenz Healthcare Services Limited. It contains necessary information and instructions about the operations of the KompleteCare web application. It also serves as a pointer for application usage and a troubleshooting document for users who come across errors or are not sure the next step to take. It is advisable that you go through this manual before or during usage of the app. Pay very close attention to the boxed items marked “important”

1.1 SYSTEM OVERVIEW

The KompleteCare web app is a web-based application accessible through most modern browsers including but not limited to Mozilla Firefox, Google Chrome and Apple Safari. It uses web technologies to present an interface where Doctors can upload patient medical records, view and update hospital appointments home consultations and perform other functions that KompleteCare presents.

1.2 ACRONYMS, ABBREVIATIONS & DEFINITIONS

App: Application

UI: User Interface

Navbar: Navigation Bar

Physician/ doctor/ practitioner: The person trained and qualified to provide medical care also referred to as Komplete care ambassador if registered on the KompleteCare platform.

Consultation: The process of history taking, physical examination and investigation by a physician/doctor/KompleteCare ambassador.
